# Sistema de Cadastro de Veículos em Java

Trabalho avaliativo desenvolvido na disciplina de Organização e Abstração na Programação.

## 📌 Descrição
Sistema simples de cadastro de veículos desenvolvido em Java, com funcionalidades de:

- Cadastro de veículos
- Listagem de veículos
- Busca por nome
- Edição de veículos
- Remoção por índice e por nome
- Ordenação alfabética (Bubble Sort)
